## 🎯 프로젝트 핵심 개념

### 베이커스 퍼센티지
밀가루를 100%로 기준 삼아 다른 재료의 비율을 나타내는 제빵 공식.

**예시**:
```
밀가루: 100% (기준)
물: 70%
소금: 2%
르방: 20%
```

### 양방향 입력 시스템
사용자가 % 또는 g 중 하나를 입력하면 나머지가 자동 계산되는 시스템.

**동작**:
- % 입력 → g 자동 계산
- g 입력 → % 역계산 + 전체 재료 재조정

### 타이머 템플릿
레시피 타입에 따라 자동으로 적용되는 타이머 단계.

**템플릿 종류**:
- **사워도우**: 16시간 과정 (긴 발효)
- **일반 도우**: 4시간 과정 (빠른 발효)
