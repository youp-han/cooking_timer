### Flutter 앱 화면 와이어프레임

이 와이어프레임은 화면의 구조, 포함될 컴포넌트, 그리고 화면 간의 이동 흐름을 정의합니다.

#### 1. 시작 및 인증 (Onboarding & Authentication)

**1.1. 로그인 화면 (Login Screen)**
*   **컴포넌트:**
    *   앱 로고 이미지
    *   "이메일" 입력 필드
    *   "비밀번호" 입력 필드
    *   [로그인] 버튼
    *   "비밀번호를 잊으셨나요?" 텍스트 링크 -> **1.3 비밀번호 재설정 화면**으로 이동
    *   "계정이 없으신가요? [회원가입]" 텍스트 링크 -> **1.2 회원가입 화면**으로 이동
    *   소셜 로그인 버튼 (구글, 애플 등)

**1.2. 회원가입 화면 (Sign-up Screen)**
*   **컴포넌트:**
    *   앱 로고 이미지
    *   "이메일" 입력 필드
    *   "비밀번호" 입력 필드
    *   "비밀번호 확인" 입력 필드
    *   [가입하기] 버튼
    *   "이미 계정이 있으신가요? [로그인]" 텍스트 링크 -> **1.1 로그인 화면**으로 이동

**1.3. 비밀번호 재설정 화면 (Password Reset Screen)**
*   **컴포넌트:**
    *   "가입하신 이메일을 입력해주세요." 안내 문구
    *   "이메일" 입력 필드
    *   [재설정 링크 보내기] 버튼

#### 2. 메인 애플리케이션 (Main Application)

로그인 성공 후 진입하는 메인 화면입니다. 하단 네비게이션 바(Bottom Navigation Bar)를 통해 주요 기능 간에 전환합니다.

**2.1. 메인 프레임 (Main Frame with Bottom Navigation)**
*   **하단 네비게이션 바:** 5개의 탭으로 구성
    *   `[계산기]` 아이콘 (기본 선택)
    *   `[타이머]` 아이콘
    *   `[내 레시피]` 아이콘
    *   `[가이드]` 아이콘
    *   `[설정]` 아이콘

---

#### 2.2. 계산기 화면 (Calculator Screen) - `[계산기]` 탭

*   **컴포넌트:**
    *   **상단 탭:**
        *   `[비율 기반]` 탭
        *   `[시간 기반]` 탭
    *   **`[비율 기반]` 탭 선택 시:**
        *   "총 스타터 양(g)" 입력 필드
        *   "비율" 입력 섹션 (스타터/밀가루/물 각각의 숫자 입력 필드)
        *   [계산하기] 버튼
        *   결과 표시 영역 (예: 스타터: 50g, 밀가루: 100g, 물: 100g)
    *   **`[시간 기반]` 탭 선택 시:**
        *   "총 스타터 양(g)" 입력 필드
        *   "준비 시간" 드롭다운 선택 메뉴 (예: 4-6시간, 6-8시간...)
        *   [계산하기] 버튼
        *   결과 표시 영역

#### 2.3. 타이머 화면 (Timer Screen) - `[타이머]` 탭 (향후 과제)

*   **컴포넌트:**
    *   **현재 진행 중인 타이머 목록:**
        *   각 목록 아이템은 "레시피 이름", "현재 단계", "남은 시간"을 표시
        *   클릭 시 -> **2.3.2 타이머 상세 화면**으로 이동
    *   **우측 하단 플로팅 버튼 `[+]`:** -> **2.3.1 새 타이머 생성 화면**으로 이동

**2.3.1. 새 타이머 생성 화면 (New Timer Screen)**
*   **컴포넌트:**
    *   "타이머 이름" 입력 필드 (예: 주말 깜파뉴)
    *   [단계 추가] 버튼
    *   **단계 목록:** (각 단계는 이름, 시간 설정 가능)
        *   단계 1: 르방 만들기 (10:00)
        *   단계 2: 오토리즈 (18:00)
        *   ...
    *   [저장 및 시작] 버튼

**2.3.2. 타이머 상세 화면 (Timer Detail Screen)**
*   **컴포넌트:**
    *   타이머 이름
    *   전체 과정 타임라인 시각화
    *   **현재 단계:** "1차 발효", "남은 시간: 01:30:00"
    *   **다음 단계:** "폴딩 #1"
    *   [일시정지] / [종료] 버튼

#### 2.4. 내 레시피 화면 (My Recipes Screen) - `[내 레시피]` 탭 (향후 과제)

*   **컴포넌트:**
    *   **저장된 레시피 목록:**
        *   각 목록 아이템은 "레시피 이름" 표시
        *   클릭 시 -> **2.4.2 레시피 상세 화면**으로 이동
    *   **우측 하단 플로팅 버튼 `[+]`:** -> **2.4.1 새 레시피 작성 화면**으로 이동

**2.4.1. 새 레시피 작성 화면 (New Recipe Screen)**
*   **컴포넌트:**
    *   "레시피 이름" 입력 필드
    *   **계산 정보 섹션:** (비율 기반 계산기 입력값 저장)
        *   총 밀가루(g), 수분율(%), 스타터 비율(%), 소금 비율(%) 등
    *   **만드는 법 섹션:**
        *   자유롭게 텍스트를 입력할 수 있는 넓은 입력 필드
    *   [저장] 버튼

**2.4.2. 레시피 상세 화면 (Recipe Detail Screen)**
*   **컴포넌트:**
    *   저장된 레시피 이름, 계산 정보, 만드는 법 텍스트 표시
    *   [이 레시피로 계산기 채우기] 버튼 -> **계산기 화면**으로 이동하며 정보 자동 입력
    *   [수정] / [삭제] 버튼

#### 2.5. 가이드 화면 (Guide Screen) - `[가이드]` 탭

*   **컴포넌트:**
    *   **상단 검색창:** "궁금한 점을 검색해보세요"
    *   **주제별 카테고리 목록:**
        *   `[피딩 비율이란?]`
        *   `[베이커스 퍼센트 이해하기]`
        *   `[자주 묻는 질문]`
        *   ...
    *   각 목록 클릭 시 -> **2.5.1 가이드 상세 화면**으로 이동

**2.5.1. 가이드 상세 화면 (Guide Detail Screen)**
*   **컴포넌트:**
    *   선택한 주제의 제목
    *   상세 설명 텍스트 및 관련 이미지

#### 2.6. 설정 화면 (Settings Screen) - `[설정]` 탭

*   **컴포넌트 (리스트 형태):**
    *   **계정 섹션**
        *   `[프로필 정보 수정]` (이름, 이메일 등)
        *   `[비밀번호 변경]`
    *   **알림 섹션 (향후 과제)**
        *   `[타이머 단계별 알림]` - 토글 스위치 (On/Off)
    *   **일반 섹션**
        *   `[테마 설정]` (라이트 모드 / 다크 모드 / 시스템 설정)
    *   **정보 섹션**
        *   `[앱 버전 정보]`
        *   `[서비스 이용약관]`
        *   `[개인정보처리방침]`
    *   `[로그아웃]` 버튼
